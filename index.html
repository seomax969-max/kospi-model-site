🟢 매수 상위 20

🔴 회피 상위 20

📐 모델 정확도 상세 — 혼동행렬

각 지표 뜻 (방향 예측을 "상승=양성/하락=음성"으로 봤을 때)
• 진양성(TP): 상승 예측 → 실제 상승 (제대로 맞춘 매수)
• 위양성(FP): 상승 예측 → 실제 하락 = 헛매수 신호
• 진음성(TN): 하락 예측 → 실제 하락 = 회피 성공
• 위음성(FN): 하락 예측 → 실제 상승 = 놓친 상승
• 정밀도(Precision) = TP/(TP+FP): "상승"이라 부른 것 중 실제 상승 비율 → 매수 신호의 신뢰도
• 특이도(Specificity) = TN/(TN+FP): 실제 하락을 하락이라 맞춘 비율 → 회피 능력
• NPV = TN/(TN+FN): "하락"이라 부른 것 중 실제 하락 비율 → 회피 신호의 신뢰도
• 민감도(Recall) = TP/(TP+FN): 실제 상승을 얼마나 포착했나
※ 절대 방향은 약함(동전던지기 부근). 이 표는 "어디서 틀리는지"를 보여주는 진단용이며, 신뢰 강점은 종목 간 순위·회피입니다.
